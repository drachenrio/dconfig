
              Dynamic Configuration Toolkit Web Demo Package List

                            Jonathan Luo

                        Last update: 5/13/2007

            http://dconfig.sourceforge.net/dconfig/webdemo.php

    This package contains Dynamic Configuration Toolkit 1.0 beta2 Web Demo.
    The Web Demo was written in JSF. Demo database uses Apache Derby database.

    Directory dconfig-db contains Apache Derby database jar files and scripts to 
    startup and shutdown the Derby database.

    Directory dconfig-webdemo contains the pre-compiled dconfig-webdemo.war file. 
    The war file can be deployed to Tomcat web server or Servlet 2.4+/JSP 2.0+ 
    compliant web servers.

    Please read individual readme.txt files under their respective directories 
    for details.
